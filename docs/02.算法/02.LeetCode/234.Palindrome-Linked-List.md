## 题目描述

> [Leetcode 地址](https://leetcode-cn.com/problems/palindrome-linked-list/)

## 类别

- 链表
- 栈

## 解题思路

### 解法一

- 切分中线，翻转对比，这里也有两种方式。

  ![](https://cdn.jsdelivr.net/gh/PPsteven/pictures/img/20200808020142.png)

  这里我们选择第一种方式，不用判断奇偶

### 代码

```python
class Solution:
    def isPalindrome(self, head: ListNode) -> bool:
        def reverse(head):
            pre, cur = None, head 
            while cur:
                cur_next_bak = cur.next 
                cur.next = pre 

                pre = cur 
                cur = cur_next_bak
            return pre 
        if not head: return True 
        slow, fast = head, head.next 
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
        mid = slow.next 
        mid = reverse(mid)

        while head and mid:
            if head.val != mid.val:
                return False
            head, mid = head.next, mid.next
        return True 
```



### 解法二【栈】

- 利用回文数，从前向后和从后向前一样的性质
- 栈可以帮助我们颠倒顺序

### 代码

```python
class Solution:
    def isPalindrome(self, head: ListNode) -> bool:
        stack = []
        p = head  # 第一次遍历是为了入栈
        while p:
            stack.append(p.val)
            p = p.next 
        p = head  # 第二次遍历是为了检测
        while p:  
            if p.val != stack.pop():
                return False 
            p = p.next 
        return True 
```
