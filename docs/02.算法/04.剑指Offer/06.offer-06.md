## 解题思路

倒叙打印链表

- 递归法
- 利用栈实现

## 代码

> 方法一：递归法

```python
class Solution:
    def reversePrint(self, head: ListNode) -> List[int]:
        if not head: return []
        return self.reversePrint(head.next) + [head.val]
```



> 方法二: 栈实现

```python
class Solution:
    def reversePrint(self, head: ListNode) -> List[int]:
        stack = []
        while head:
            stack.append(head.val)
            head = head.next 
        return stack[::-1]
```
