## 注释

- 206 Partial Content

  包含 Content-Range 字段指明请求资源的范围

- 304 Not Modified

  允许访问资源，但是不符合附加条件。
  附加条件有：

  - If-Match，If-ModifiedSince，If-None-Match，If-Range，If-Unmodified-Since
  - If-XXX 标识 **附加条件**
