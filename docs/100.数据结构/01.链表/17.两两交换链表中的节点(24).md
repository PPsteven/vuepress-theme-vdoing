## Python 代码

```python
class Solution:
    def swapPairs(self, head: ListNode) -> ListNode:
        if not head or not head.next:
             return head 
        after = head.next # [2]
        head.next = self.swapPairs(after.next) # [1] -> recur([2]) 所以递归的结果应该是 [4] 
        after.next = head # [2] -> [1]
        
        return after # [4] 对应的是下一层的 [2] 的位置
```
