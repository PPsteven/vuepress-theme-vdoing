## 题目描述

## 样例

```
Input: [[10,16],[2,8],[1,6],[7,12]]
Output: 2
# x=6,11 射出两根箭能把所有气球引爆
```

## 题解

## Python代码示例

```python
class Solution:
    def partitionLabels(self, S: str) -> List[int]:
        last = {}
        for idx, s in enumerate(S):
            last[s] = idx
        start, end = 0, 0
        label = []
        for idx, s in enumerate(S):
            end = max(end, last[s])
            if idx == end: # 走到end，划分字母
                label.append(end - start + 1)
                start = idx + 1 # 开新的区间           
        return label 
```
